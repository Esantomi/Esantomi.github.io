깃랩(GitLab) 등을 공부하려고 하니, 데브옵스(DevOps), CI/CD 같은 개념들을 짚고 넘어가지 않을 수 없게 되었다. 그래서 이 참에 한번 정리해 보려고 한다.

## CI (Continuous Integration)

the practice of automating the integration of code changes from multiple contributors into a single software project.

![image](https://user-images.githubusercontent.com/61646760/130063610-174e963e-1dd4-4fa5-9378-9c46eedd01d0.png)

It’s a primary DevOps best practice, allowing developers to frequently merge code changes into a central repository where builds and tests then run.
